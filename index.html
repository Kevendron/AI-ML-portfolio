Fashion Object Image Classification

his project focuses on multi-class image classification of fashion objects using both traditional machine learning and deep learning approaches. The objective was to analyze a dataset of 50,000 labeled clothing images across 10 categories and compare model performance across different algorithmic paradigms. Three models were implemented and benchmarked: Convolutional Neural Network (CNN) Support Vector Machine (SVM) Random Forest Classifier The CNN model achieved 94% classification accuracy, outperforming traditional ML models and reducing misclassification errors by approximately 10%.
